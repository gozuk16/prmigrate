# GitHub トークン管理を gh 認証ストアに委譲 実装計画

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** `github.token` を config に書かなくても `gh auth login` 済みであれば動作するよう、トークン解決に `cli/go-gh` フォールバックを追加する。

**Architecture:** `internal/config/config.go` の `ResolveSecrets()` を戻り値 `error` に変更し、既存の TOML / env var に加えて `github.com/cli/go-gh/v2/pkg/auth.TokenForHost()` を第3の手段として追加する。`cmd/prmigrate/main.go` でエラーハンドリングを追加する。その他のパッケージは無変更。

**Tech Stack:** Go 1.22、`github.com/cli/go-gh/v2`（`pkg/auth` のみ使用）

## Global Constraints

- ブランチ: `feature/gh-auth`
- `internal/githubapi/`・`internal/githubimport/`・`internal/pipeline/` は変更しない
- トークン解決の優先順位: TOML config → `PRMIGRATE_GITHUB_TOKEN` 環境変数 → `cli/go-gh` 認証ストア
- すべて空の場合のエラーメッセージ: `github token not found: set github.token in config, PRMIGRATE_GITHUB_TOKEN env var, or run "gh auth login"`
- `githubHostname("https://api.github.com")` → `"github.com"`（`api.` プレフィックスを除去）
- `githubHostname("https://github.example.com/api/v3")` → `"github.example.com"`（そのまま）

```go
// GithubHostname extracts the gh-compatible hostname from a GitHub API base URL.
// "https://api.github.com"           -> "github.com"
// "https://github.example.com/api/v3" -> "github.example.com"
func GithubHostname(apiBase string) string {
	u, err := url.Parse(apiBase)
	if err != nil || u.Host == "" {
		return "github.com"
	}
	host := u.Hostname()
	if strings.HasPrefix(host, "api.") {
		return strings.TrimPrefix(host, "api.")
	}
	return host
}
```

`ResolveSecrets()` を置き換える（Bitbucket トークンも引き続きここで解決する）：

```go
// ResolveSecrets resolves tokens in priority order:
//  1. Value already set in the TOML config field
//  2. PRMIGRATE_BITBUCKET_TOKEN / PRMIGRATE_GITHUB_TOKEN environment variables
//  3. gh CLI credential store (GitHub only, via cli/go-gh)
//
// Returns an error if the GitHub token cannot be resolved by any means.
// Callers that run in dry-run mode may ignore the returned error.
func (c *Config) ResolveSecrets() error {
	if c.Bitbucket.Token == "" {
		c.Bitbucket.Token = os.Getenv("PRMIGRATE_BITBUCKET_TOKEN")
	}
	if c.GitHub.Token == "" {
		c.GitHub.Token = os.Getenv("PRMIGRATE_GITHUB_TOKEN")
	}
	if c.GitHub.Token == "" {
		host := GithubHostname(c.GitHub.APIBase)
		token, _ := auth.TokenForHost(host)
		c.GitHub.Token = token
	}
	if c.GitHub.Token == "" {
		return fmt.Errorf("github token not found: set github.token in config, PRMIGRATE_GITHUB_TOKEN env var, or run \"gh auth login\"")
	}
	return nil
}
```
